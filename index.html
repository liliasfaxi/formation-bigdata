    Formation Big Data¶

    Overview¶

    Ces travaux pratiques permettront de manipuler deux des technologies phares de l'écosystème Big Data: Hadoop et Spark.

    Ils seront répartis comme suit:

    • Partie 1: Hadoop HDFS
    • Partie 2: Hadoop Map Reduce
    • Partie 3: Traitement par Lot avec Spark
    • Partie 4: Traitement en Streaming avec Spark

    Outils et Versions¶

    • Apache Hadoop Version: 3.3.6.
    • Apache Spark Version: 3.5.0
    • Docker Version latest
    • IntelliJ IDEA Version latest (ou tout autre IDE de votre choix)
    • Java Version 1.8.
